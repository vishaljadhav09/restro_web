# Ya Basa Restaurant – Warm Brown & Red Theme

Next.js 15 website using the **Burger King-inspired warm color palette**:
- `#885133` Warm Brown — primary brand
- `#f4f0ec` Cream — background  
- `#ffffff` White — cards/surfaces
- `#d62300` Bold Red — CTAs & accents
- `#f0e6d7` Warm Beige — section backgrounds

## Run Locally
```bash
npm install && npm run dev
```
